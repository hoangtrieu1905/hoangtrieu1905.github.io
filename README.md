# hoangtrieu1905.github.io

Personal portfolio website for Hoang Trieu — Mathematics & Computer Science @ Augustana College, incoming PhD student in AMSC @ University of Maryland.

## Structure

```
index.html      — main single-page site
style.css       — all styles
script.js       — scroll effects, mobile nav, fade-in animations
assets/
  profile.jpg   — profile photo (replace placeholder)
  cv.pdf        — downloadable CV (add your own)
```

## Image placeholders

Replace the following before publishing:
- `assets/profile.jpg` — your profile photo (square, min 400×400px recommended)
- `assets/cv.pdf` — your CV PDF
